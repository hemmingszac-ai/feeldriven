# FieldDriven

A minimal productivity SaaS starter built with Next.js.

## What this includes

- Next.js app with TypeScript support
- Simple responsive landing page
- Placeholder API route ready for custom backend logic

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open `http://localhost:3000`

## Environment

Add your OpenRouter API key to `.env` as:

```env
OPENROUTER_API_KEY=sk-or-v1-...your-key...
```
